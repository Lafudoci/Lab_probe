#ProbeCube

##專案簡介
ProbeCube是個基於開放硬體的物聯網專案，動手製作WiFi連網的機器來偵測空氣品質並組織共同的觀測地圖。

##如何開始
1. 準備材料
2. 焊接或使用麵包板連接各部元件
3. 註冊[Thingspeak](https://thingspeak.com/)帳號並設定上傳channel
4. 將程式碼客製修改後編譯寫入Arudino Nano
5. 到ProbeCube共同觀測地圖提交你的channel ID分享資料(開發中)

##連結
